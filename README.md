# Zgadnij-liczbe-2

Zgadnij Liczbę 2 to konsolowa gra napisana w języku C#. Celem gracza jest odgadnięcie wylosowanej przez komputer liczby przy jak najmniejszej liczbie prób, w jak najkrótszym czasie. Po każdej nieudanej próbie gracz otrzymuje wskazówkę informującą, czy szukana liczba jest większa czy mniejsza od podanej.

Program oferuje różne poziomy trudności oraz dodatkowy tryb New Game Plus, w którym liczba jest okresowo losowana ponownie, co zwiększa poziom trudności rozgrywki.
